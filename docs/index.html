Visão Geral

Descrição

O BID System é uma aplicação web desenvolvida para gerenciamento de contratos, ordens de serviço e faturamento. O sistema oferece uma interface moderna e intuitiva, construída com foco em performance e usabilidade.

Objetivos

  • Gerenciamento eficiente de contratos e ordens de serviço
  • Controle de usuários e permissões
  • Interface responsiva e moderna
  • Performance otimizada

Stack Tecnológico

Frontend

  • HTML5
  • CSS3 (Bootstrap 4.6)
  • JavaScript (ES6+)
  • AdminLTE 3.2

Backend

  • PHP 8.1
  • MySQL 8.0

Guia de Instalação

Requisitos do Sistema

  • PHP 8.1 ou superior
  • MySQL 8.0 ou superior
  • Apache/Nginx com mod_rewrite
  • Composer (Gerenciador de dependências PHP)
  • Node.js e NPM (opcional, para desenvolvimento)

Passo a Passo

1. Clonar o Repositório


git clone https://github.com/seu-usuario/bid-system.git
cd bid-system

2. Configurar o Ambiente


# Copiar arquivo de configuração
cp .env.example .env

# Editar configurações
nano .env

3. Instalar Dependências


# Instalar dependências PHP
composer install

# Instalar dependências frontend (opcional)
npm install

4. Configurar Banco de Dados


-- Criar banco de dados
CREATE DATABASE bid_system;

-- Importar estrutura
mysql -u root -p bid_system < database/structure.sql

-- Importar dados iniciais
mysql -u root -p bid_system < database/seeds.sql

Gestão de Usuários

Sistema completo de gerenciamento de usuários com as seguintes operações:

CRUD de Usuários
Create (Criar)

// Exemplo de criação de usuário
async criarUsuario(usuario) {
    try {
        const response = await Api.Usuarios.criar(usuario);
        if (response.sucesso) {
            await this.loadUsuarios();
            Notify.success('Usuário cadastrado com sucesso!');
        }
    } catch (error) {
        Notify.error('Erro ao cadastrar usuário');
    }
}
Read (Ler)

// Exemplo de listagem de usuários
async loadUsuarios() {
    const response = await Api.Usuarios.listar();
    if (response.sucesso) {
        this.usuarios = response.dados;
        this.renderTable();
    }
}
Update (Atualizar)

// Exemplo de atualização de usuário
async atualizarUsuario(id, usuario) {
    const response = await Api.Usuarios.atualizar(id, usuario);
    if (response.sucesso) {
        await this.loadUsuarios();
        Notify.success('Usuário atualizado com sucesso!');
    }
}
Delete (Excluir)

// Exemplo de exclusão de usuário
async delete(id) {
    if (confirm('Confirma exclusão?')) {
        const response = await Api.Usuarios.excluir(id);
        if (response.sucesso) {
            await this.loadUsuarios();
            Notify.success('Usuário excluído com sucesso!');
        }
    }
}

Dashboard

O Dashboard é a tela principal do sistema, apresentando uma visão geral das informações mais relevantes.

Funcionalidades

1. Calendário

Calendário interativo para visualização de eventos como contratos, ordens de serviço e faturas.

Tecnologias Utilizadas
  • HTML5
  • CSS3 (com classes utilitárias do Tailwind e Bootstrap)
  • JavaScript (Vanilla/ES6+)
Características
  • Visualização mensal
  • Indicadores visuais por tipo de evento:
    • Contratos (azul)
    • Ordens de Serviço (verde)
    • Faturas (vermelho)
  • Navegação entre meses
  • Destaque para o dia atual
  • Modal para detalhes dos eventos
  • Design responsivo
  • Animações suaves
Estrutura de Arquivos

bid/
├── assets/
│   ├── css/
│   │   └── calendar.css    # Estilos específicos do calendário
│   └── js/
│       └── calendar.js     # Classe Calendar e funcionalidades
└── pages/
    └── dashboard.html      # Implementação do calendário
Implementação

O calendário é implementado através de uma classe JavaScript que gerencia:

  • Renderização do grid de dias
  • Gerenciamento de eventos
  • Navegação entre meses
  • Interatividade com usuário
Dependências
  • Bootstrap 5 (estilos base e componentes)
  • Tailwind CSS (utilitários e customizações)
  • Font Awesome (ícones)
Manutenção

Para adicionar ou modificar eventos, utilize o objeto events na classe Calendar:


this.events = {
    'YYYY-MM-DD': { 
        type: 'contrato|os|fatura', 
        title: 'Descrição do Evento' 
    }
};
