Visão Geral

O BID System é um sistema de gerenciamento de manutenções preventivas desenvolvido para otimizar e controlar processos de manutenção em equipamentos e instalações. O sistema oferece uma solução completa para planejamento, execução e acompanhamento de manutenções preventivas.

Objetivos Principais

  • Gerenciar cronogramas de manutenção preventiva
  • Controlar ordens de serviço de manutenção
  • Acompanhar histórico de manutenções
  • Gerenciar equipes e recursos
  • Gerar relatórios e indicadores de desempenho

Stack Tecnológico

Frontend

  • HTML5
  • JavaScript (ES6+)
  • AdminLTE 3 (Bootstrap)
  • Tailwind CSS
  • Bibliotecas: Axios, jQuery, Toastr, Chart.js

Backend

  • PHP 8.3.13
  • MySQL
  • RESTful API

Autenticação

    "email": "string",
